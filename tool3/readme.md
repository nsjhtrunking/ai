# 主題
1. 幫我建立互動網頁，需要包含 css , java , 包成單個 html 檔，可電腦瀏覽器或手機使用，內容是
	1. 班上預設30位同學可自行輸入學生學號和姓名, 後面欄位包含每節課學生的表現紀錄, 比如出缺曠, 上課行為,吵鬧或睡覺,未帶課本等.
	1. 每個欄位都可以點按, 最後輸出每節課, 紀錄並生成統計圖表.
# UI 
1. 頁面支援電腦瀏覽器及手機平板，電腦瀏覽器支援滑鼠滾輪縮放，單點左鍵拖曳、手機平板支援雙指觸控拖曳、縮放以及單指點選
	1. 左方分類班級,右方呈現所有學生紀錄
