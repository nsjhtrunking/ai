排序動畫範例

這是一個簡化版的排序視覺化示意。點擊「打亂」產生新的數列，再按「開始排序」即可看到資料列如何逐步被重排。 真實使用時，你可以把這個頁面換成自己的 Java Applet / JS 轉寫的視覺化程式。

說明與延伸

目前示意只用最基本的氣泡排序（Bubble Sort）。如果要教學，可以：

  • 把每一回合的比較次數標在畫面上。
  • 切換不同演算法（選擇排序、插入排序、快速排序）。
  • 讓學生去預測「下一步會交換哪兩個柱子」。
